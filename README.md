# Rare Planet Clone 🌏

## 📌 Project Overview

The **Rare Planet Clone** is a simple frontend website inspired by the Rare Planet platform.
This project recreates the design and layout of the website using **HTML and CSS only**.
It displays handcrafted lifestyle products inspired by Indian art and culture.

This project helps in understanding **basic web development concepts such as page layout, styling, and responsive design**.

---

## 🎯 Objectives

* Practice **HTML structure and semantic elements**
* Improve **CSS styling and layout design**
* Build a **static e-commerce website UI**
* Understand how product pages and navigation are organized

---

## ✨ Features

* Navigation Bar for easy page access
* Homepage with banner and featured products
* Product section with images, names, and prices
* Clean and simple user interface
* Responsive layout using CSS

---

## 🛠 Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout design
